## STEP 1: SUPABASE TABLES

Create these tables in Supabase. If a Supabase project is not yet connected, connect one first.

### polls table

```sql
CREATE TABLE polls (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  question TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  category TEXT NOT NULL CHECK (category IN ('technology_ai', 'culture_society', 'business', 'cities', 'identity', 'economy', 'women_equality', 'arts_expression')),
  options JSONB NOT NULL DEFAULT '[]',
  poll_type TEXT DEFAULT 'card' CHECK (poll_type IN ('hero', 'card', 'strip')),
  status TEXT DEFAULT 'draft' CHECK (status IN ('draft', 'live', 'closed')),
  is_featured BOOLEAN DEFAULT false,
  tags TEXT[] DEFAULT '{}',
  total_votes INTEGER DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now(),
  updated_at TIMESTAMPTZ DEFAULT now()
);

CREATE INDEX idx_polls_category ON polls(category);
CREATE INDEX idx_polls_status ON polls(status);
CREATE INDEX idx_polls_slug ON polls(slug);
```

The `options` JSONB stores an array of objects like:

```json
[
  {"text": "Option A", "votes": 0},
  {"text": "Option B", "votes": 0},
  {"text": "Option C", "votes": 0},
  {"text": "Option D", "votes": 0}
]
```

### votes table

```sql
CREATE TABLE votes (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  poll_id UUID REFERENCES polls(id) ON DELETE CASCADE,
  option_index INTEGER NOT NULL,
  fingerprint TEXT NOT NULL,
  country_code TEXT DEFAULT 'XX',
  utm_source TEXT,
  created_at TIMESTAMPTZ DEFAULT now(),
  UNIQUE(poll_id, fingerprint)
);

CREATE INDEX idx_votes_poll_id ON votes(poll_id);
CREATE INDEX idx_votes_fingerprint ON votes(fingerprint);
```

### email_captures table

```sql
CREATE TABLE email_captures (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  email TEXT NOT NULL,
  poll_id UUID REFERENCES polls(id),
  source TEXT DEFAULT 'share_gate',
  created_at TIMESTAMPTZ DEFAULT now()
);
```

### Row Level Security

```sql
ALTER TABLE polls ENABLE ROW LEVEL SECURITY;
ALTER TABLE votes ENABLE ROW LEVEL SECURITY;
ALTER TABLE email_captures ENABLE ROW LEVEL SECURITY;

-- Anyone can read live polls
CREATE POLICY "Public can read live polls" ON polls FOR SELECT USING (status = 'live');

-- Anyone can insert votes (rate limiting handled in application)
CREATE POLICY "Public can vote" ON votes FOR INSERT WITH CHECK (true);

-- Anyone can read votes (for result counts)
CREATE POLICY "Public can read votes" ON votes FOR SELECT USING (true);

-- Anyone can submit email
CREATE POLICY "Public can submit email" ON email_captures FOR INSERT WITH CHECK (true);
```

### Enable Realtime on polls table

In Supabase dashboard: Database > Replication > Enable realtime for the `polls` table. This powers the live vote counter.

## STEP 3: INSTALL FINGERPRINTJS

```bash
npm install @fingerprintjs/fingerprintjs
```

Create a utility at `src/lib/fingerprint.ts`:

```typescript
import FingerprintJS from '@fingerprintjs/fingerprintjs';

let fpPromise: Promise<any> | null = null;

export async function getFingerprint(): Promise<string> {
  if (!fpPromise) {
    fpPromise = FingerprintJS.load();
  }
  const fp = await fpPromise;
  const result = await fp.get();
  return result.visitorId;
}
```

## STEP 4: VOTE SUBMISSION API

Create an API route (or Supabase Edge Function) at `/api/vote` that handles:

```
POST /api/vote
Body: { poll_id, option_index, fingerprint, country_code }

Logic:
1. Check if a vote already exists for this poll_id + fingerprint combination
2. If duplicate: return { error: 'Already voted', status: 409 }
3. If new: INSERT into votes table
4. UPDATE the polls table: increment options[option_index].votes by 1 and total_votes by 1
5. Return { success: true, updated_options, total_votes }
```

Use Supabase RPC or a direct update query to atomically increment the vote count:

```sql
-- Create this function in Supabase SQL editor
CREATE OR REPLACE FUNCTION increment_vote(p_poll_id UUID, p_option_index INTEGER)
RETURNS void AS $$
BEGIN
  UPDATE polls
  SET
    options = jsonb_set(
      options,
      ARRAY[p_option_index::text, 'votes'],
      (COALESCE((options->p_option_index->>'votes')::int, 0) + 1)::text::jsonb
    ),
    total_votes = total_votes + 1,
    updated_at = now()
  WHERE id = p_poll_id;
END;
$$ LANGUAGE plpgsql;
```
